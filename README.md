# SEMOMO

# I.프로젝트 정보
<br>
<br>

**1. 소개**
> 모임끼리 소통하기 쉬운 '소모임 플랫폼'
<br>
<br>

**2. 제작기간**
> 기간 : 2023.10.01 ~ 2023.10.24 (3주)
<br>
<br>

**3. 주요기능**
>+ 소모임 <br>
>    + 소모임 개설
>    + 소모임 가입 승인
>    + 모임 장소(지도로 표시),날짜,시간
>    + 모임원 참가or취소
>    + 관심 모임 저장<br>
>+ 소모임 內 게시판 <br>
>    + 사진첩 및 게시판 작성
>    + 게시판 댓글 작성
>    + 댓글 內 좋아요 버튼 기능<br>
>+ 개인 피드 <br>
>    + 유저끼리 팔로우/팔로잉 기능 
>    + 피드 작성
>    + 피드 댓글 작성
>    + 댓글 內 좋아요 버튼 기능<br>
>+ 통합검색 시, 소모인 & 피드 동시 확인
>+ 참여중인 모임의 채팅방 이용 기능 구현
>+ 관리자 : 신고 당한 모임 관리 / 블랙 회원 관리 / 공지사항 게시판
>+ 마이페이지 <br>
>    + 내 정보 수정 
>    + 내가 참여하고 있는 모임 조회
>    + 내가 찜한 모임 조회
>    + 관리자 페이지
>    + 내 피드 조회<br>
<br>
<br>
